## 🔍 직면한 문제 & 해결 (Troubleshooting)
- **문제:** ROS2 humble을 설치하려고 했으나, Humble 경로가 없다고 떴다.
- **원인:** ROS2 Humble은 Ubuntu 22.04용이다.그래서 경로를 못 찾았음.
- **해결:** 대신에 ROS2 Jazzy를 설치해주자.

- **문제:** `pip install -U colcon-common-extensions` 명령어로 colcon을 다운로드 받으려 했다. 그러나 에러가 떴다.
- **원인:** Ubuntu 23.04 버전부터 파이썬의 보안 정책(PEP 668) 때문에 발생한다. 리눅스 시스템은 자기 시스템을 돌리기 위해 파이썬을 아주 중요하게 생각한다. 사용자가 `pip`로 패키지를 마구 설치하다가 시스템이 사용하는 파이썬 파일을 덮어버리면 OS 자체가 망가질 수 있다. 그래서 최신 Ubuntu는 시스템에 파이썬을 직접 `pip install`하는 대신 전용 패키지 관리자 (`apt`)를 쓰거나 가상 환경을 만들라는 경고를 보낸다.
- **해결:** ROS2를 공부하고 있다면, `pip` 대신 우분투 공식 저장소에 등록된 `apt` 패키지를 사용하는 것이 가장 깔끔하고 안전하다.
```bash
sudo apt update
sudo apt install python3-colcon-common-extensions
```
